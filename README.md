# CLV-Prediction
A UK-based online retailer sells to ~4,300 customers across 37
countries. Without knowing which customers are most valuable, marketing
budget is spread evenly — wasting spend on low-value customers while
under-investing in high-value ones.

This project builds a regression model that predicts each customer's
next 90-day revenue, then segments them into three actionable tiers:

  High CLV  →  top 33% — priority retention, dedicated outreach
  Mid CLV   →  middle 33% — highest ROI for upsell campaigns
  Low CLV   →  bottom 33% — automated touchpoints only
